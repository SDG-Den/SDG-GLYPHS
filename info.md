Package Name: sdg-glyphs
Descriptive Name: SDG Pen/Rune Input System
Source: https://github.com/SDG-Den/SDG-GLYPHS
Maintainer: SDGDen
Version:0.2

Dependencies: 
python3-gobject, python3-cairo, wtype, python3 (>=3.10)
Optional: tesseract-ocr, tesseract-data-eng

Description: 
Pen/stylus-based glyph (rune-gesture) input system for Wayland compositors. Users draw rune-shaped strokes on a transparent fullscreen overlay to type text or execute shell commands. Features 71 built-in glyphs, DTW-based recognition, recording modes, and GTK3 Cairo rendering.
